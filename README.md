# java-web-api-maven-spark-hello-world

## Description
A POC for java spark webframework.

## Tech stack
- java
- maven
  - spark

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
